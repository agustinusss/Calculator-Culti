# Cultivation Simulator Tools (Calculator-Culti)

Welcome to **Cultivation Simulator Tools**, a suite of premium tools and calculators designed for the Roblox game *Cultivation Simulator*. 

## 🌐 Live Demo

You can access the hosted version of the website here:  
**[https://agustinusss.github.io/Calculator-Culti/](https://agustinusss.github.io/Calculator-Culti/)**

## 🛠️ Features

This project provides various tools to help you optimize your gameplay in Cultivation Simulator:

- **Wing Calculator**: Calculate Wing Coefficient and ATBM%.
- **Elixir Calculator**: Plan and optimize your elixir usage.
- **World Rate Calculator**: Calculate various world rates.
- **Wing Simulator**: Simulate wing upgrades and stats.
- **Reroll Simulator**: Simulate rerolls to test your luck and strategies.
- **Absorb Analysis**: Analyze absorption stats.
- **Pet Absorb Analysis**: Calculate required elixirs based on your Pet's potential (HP and Atk).

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your web browser. No complex build tools or dependencies are required.

```bash
git clone https://github.com/agustinusss/Calculator-Culti.git
```
Then, open the `index.html` file in your preferred web browser.

## 💻 Technologies Used

- **HTML5**: For the structure of the tools and calculators.
- **CSS3**: Vanilla CSS for styling, utilizing a glassmorphism aesthetic and modern layout.
- **JavaScript**: For the logic and calculations behind each tool.
